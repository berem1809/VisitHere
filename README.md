# Demo :movie_camera:
## Developer Portfolio

Clean React portfolio with sections for landing, about, skills, experience, projects, blog, services, testimonials, and contact.

### Use It (Fork)
1) Fork this repo to your GitHub. 
2) Clone your fork:
```bash
git clone https://github.com/<your-username>/<your-fork>.git
cd <your-fork>
npm install
npm start
```
3) Update data and assets, then `npm run build` to deploy.

### What to Edit
- Text/content: [src/data](src/data)
- Images/PDF/fonts: [src/assets](src/assets)
- Theme colors: [src/data/themeData.js](src/data/themeData.js)

### Scripts
- `npm start` — dev server
- `npm run build` — production bundle

### Deploy
Host the `build` folder on Netlify, Vercel, or GitHub Pages. `public/_redirects` is included for single-page routing.
